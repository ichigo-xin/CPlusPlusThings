# const

## 定义常量
必须给定初始值

## const和指针
- 指向常量的指针  
  const int *p p所指向的值不能通过p来修改
- 常指针  
  int * const p p所包含的地址是不变的，即指针不变
- 指向常量的常指针
  
## const和函数
主要是const修饰指针参数和引用  
void fun1(const A *a)  
void fun2(const A &a)  
目的：为了不会在函数内部改变传入参数的值，同时可以节省空间。  
  
对于非内部数据类型作为输入参数，应当使用“const引用”方式，提高效率，而对于内部数据类型，则直接使用值传递就行。
